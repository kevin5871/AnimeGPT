 九尾狐活动 5上








フォリニック :  ……歌声？
フォリニック :  どうして……

スズラン :  街中に響き渡ってますね……


青白い薄暮に覆われた街並みの、点々とした明かりの間で、人々が生活している。
二人の眼前には、人っ子一人いない街が広がっていた。朧げな歌声が一枚の網のように、通りを覆う。その向こうには、夕日が沈もうとしていた。




剥き出しになった源石群が、暴れまわる抗議者が、蒼白の星空とレンガが、静寂に沈んでいく。
歌声が……ただ歌声だけが残された……






フォリニック :  どういうこと……今見えたのは何？
フォリニック :  そしてこの歌は……民謡？　それとも何かの詩？　夕暮れ時に歌なんて……
フォリニック :  ウォルモンドにはそんな風習があるの？

スズラン :  えっ……フォリニックお姉さん、歌詞が聞き取れるんですか？

フォリニック :  いいえ……でもこれは、リターニア語？　現代のリターニア語にはこんな曖昧な発音はないはず……

スズラン :  ……とても、きれいな歌ですね。
スズラン :  初めてロドスのブリッジから大地を眺めたときを思い出します。どこまでも広がっていて……古めかしくて……
スズラン :  これは……アーツです……極東の古代アーツと同じです！














セベリン :  探せ。

住民 :  わかりました。見つけた後は？　直接逮捕しますか？

セベリン :  ああ。

住民 :  ですが……捕らえれば、何かを企んでいる奴らに、扇動の口実に使われてしまうかもしれません。

セベリン :  ……構うな。逮捕は彼らが、感染者と非感染者の対立を煽ることを防ぐためでもある。

スズラン :  セベリン憲兵長！　こ、この歌は一体？

セベリン :  ……冬霊の歌だ。古代の儀式や葬儀でしか用いられないものだ。
セベリン :  感染者と非感染者の対立を後押しし、危険な者たちを率いて街の動力炉を破壊した諸悪の根源は、冬霊人だろう。
セベリン :  寛容な措置などと言っている場合ではない、これは何かの信号だ。
セベリン :  冬霊人が何の目的もなく身を明かすことはない。必ず裏で動いている者がいる。それも術師の素質を持った者たちだ、気をつけろ。

フォリニック :  ――冬霊人というのは一体何者なんですか？

セベリン :  この場所に住まうキャプリニー族だ。数百年前、彼らこそがこの群山の住民だった。

フォリニック :  いったい、過去に何があったんですか？

セベリン :  ……ウォルモンドと冬霊人が衝突した、それだけだ。

フォリニック :  いいえ、違う。それだけじゃないはずです。
フォリニック :  ただの植民地問題であれば、クルビアにもレム・ビリトンにも、その他のどの国家にも存在します。
フォリニック :  でも今回は明らかに問題が大きい。衝突があっただけなんて……そんな簡単に話をはぐらかせると思いましたか？

セベリン :  ……君たちに話すべきか判断できないんだ。これは間違いなくロドスが知るべき範疇を超えている……
セベリン :  一人の憲兵がおおっぴらに余所の企業に教えたとなれば、一部の貴族たちが不愉快な思いをすることになるだろう。

フォリニック :  貴族……？　不愉快？？


フォリニック :  ――アントはあなたたちのために犠牲になったのよ！　いつまで体面を気にしてるつもりよ！


スズラン :  フォ、フォリニックお姉さん！　尊い犠牲を……反論するときの理由に使っちゃダメです。
スズラン :  セベリン憲兵長……！　ひとまず今やろうとしている捜査を止められますか？　皆さんが嫌な思いをすれば、取り返しのつかない事態になるかもしれません！

フォリニック :  ――！
フォリニック :  ……いや、唯一の手がかりが目の前にあるのに、見過ごすわけにはいかないわ。

スズラン :  えっ……？

フォリニック :  セベリンさん、リターニアには多くの移動都市があります。あの塔から高みの見物をしている貴族たちが、あなたたちの生死を気にしていると、本当に思いますか？
フォリニック :  大裂溝の被害を受けたとしても、一番近くを運行している移動都市が支援に駆けつけてくれたなら、飢饉なんて起きないんです。
フォリニック :  セベリンさん、あなたたちの本当の味方は誰か、ちゃんと見定めてください！

セベリン :  わかっているさ……フォリニックさん、そんなことは、私が誰よりもはっきりと理解している。
セベリン :  だがロドスは、今後永続的に支援をしてくれる訳ではないだろう。今回の事件が解決し、ウォルモンドがあるべき姿に戻った後、あの高い塔からの通告書を受け取るわけにはいかないんだ。
セベリン :  ……ご理解いただきたい。

スズラン :  フォリニックお姉さん、私たちは――










スズラン :  ――爆発！？　また襲撃ですか？

住民 :  憲兵長、憲兵長！　助けてくれ！

セベリン :  落ち着け、どうした？

住民 :  奴らが、奴らが、あの市街防衛装置を起動させた！　爆発して、燃えて、奴らが――！

セベリン :  落ち着け！　冷静になれ！


タチヤナ :  憲兵長！　ドデカフォニー通りを巡回していた憲兵が、全員感染者たちに捕まってしまったようです！　一切連絡がつきません！
タチヤナ :  感染者たちは、あの蓄音機を使って頑なに抵抗を続けています。もう街全体が暴乱状態です――！

セベリン :  ……本来の彼らならば、そこまで組織的には動けないはずだ、誰が指揮を執っている？

タチヤナ :  まだわかりません――ですが多くの非感染者も抗議者集団に加わったようです！　……ウォルモンドは既に……真っ二つです！

セベリン :  チッ、冬霊人め……そっちがそのつもりなら。
セベリン :  街全体を包囲し、彼らを抑え込め。なんとしても街を取り戻す――
セベリン :  ――ゴホッ、ゴホゴホッ！

タチヤナ :  憲兵長！

セベリン :  クソッ……君は先に議事堂に戻り、警報を鳴らせ。これを単なる感染者の抗議活動だとは思うな……
セベリン :  ウォルモンド全体に戒厳令を敷き、民兵を集めるんだ。

タチヤナ :  でも……もし殺し合いに発展してしまえば、ウォルモンドが冬を越すことは不可能になります――！

セベリン :  言われた通りにやれ！

タチヤナ :  はっ、はい！

セベリン :  ……

セベリン :  ロドスのお二人……君たちは、自分たちが正しいと思うようにやればいい。
セベリン :  正直なところ、私は君たちを見張っている余裕などないのでな。

フォリニック :  ではもっと多くの情報を――

スズラン :  で、でも今は先に救援のお手伝いをしないと……きっと何の罪もない人たちもたくさん巻き込まれてます……

フォリニック :  ――私たちは犯人を見つけ出します。全てが終わった後に、必ず。

セベリン :  ……ウォルモンドは、ロドスが満足のいく答えを提示すると約束しよう。

フォリニック :  その約束を守っていただけることを、切に願います。

セベリン :  それまで私が生きていればな。








スズラン :  ……フォリニックお姉さん、セベリン憲兵長を止めるべきだったのではないでしょうか。

フォリニック :  無駄よ、見たでしょう。彼らはみんな、私たちの話を聞く余裕なんてない。

スズラン :  そ……そうですが……
スズラン :  ですがもし憲兵長の言った通り、飢饉が目前に迫っているのなら、私たちは……

フォリニック :  ……早く犯人を見つけ出さないとマズイわね。この暴乱を抑えるためにも、手がかりは手放さないようにしなくちゃ。

スズラン :  そういう意味じゃありません……
スズラン :  ……
スズラン :  フォリニックお姉さん、犯人を見つけたら、本当に問題が全部解決すると思っているんですか？

フォリニック :  えっ、私は――

スズラン :  ――お姉さん、冷静になってください。

フォリニック :  ……スズラン？

スズラン :  お姉さんだってわかってるはずです……今は皆さん頭に血が上っていて、いつもの皆さんとは違うって。
スズラン :  私はこう思います……あ、あくまで私の予想ですよ。でも私の予想は意外と当たるんです！
スズラン :  えっと、もしかしたら……今この瞬間、もっと悪い何かが起こり始めているのではないでしょうか。正しいとか間違っているとか、そんな言葉では説明できないような何かが……
スズラン :  だから、状況がもっと悪くなる前に私たちがやるべきことは、巻き込まれた罪のない住民たちを少しでも助けることなんじゃないかと思います。
スズラン :  フォリニックお姉さん……もし今ここでセベリン憲兵長を止めなければ、本当に取り返しのつかないことになってしまいます。

フォリニック :  ……そうかもしれないわね。

スズラン :  （何か言いかけてやめる）……

スズラン :  ……はい。
