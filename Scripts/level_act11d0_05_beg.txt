 九尾狐活动 5上








폴리닉 :  ……노랫소리?
폴리닉 :  어떻게……

스즈란 :  온 거리에 노랫소리가 울려 퍼지고 있어요……


하늘색 황혼이 물든 적막한 거리의 드문드문한 불빛 사이에서, 사람들은 살아가고 있다.
두 사람의 눈앞에 펼쳐진 거리는 텅 비어버렸고, 흐릿한 노랫소리는 한 장의 그물이 되어 석양을 끌어내렸다.




훤히 드러난 오리지늄 덩어리, 날뛰던 시위자들, 그리고 창백한 밤하늘과 벽돌까지… 모두 적막에 잠겨버렸다.
노랫소리…… 남은 건 오직 노랫소리뿐……






폴리닉 :  이게 무슨…… 내가 지금 뭘 본 거지? 이건…… 아츠인가?
폴리닉 :  이건…… 민요? 아니면 시인가? 노을이 지는 때에?
폴리닉 :  월루몽드에 이런 풍습이 있었나?

스즈란 :  에엣…… 폴리닉 언니, 가사를 알아들을 수 있는 거예요?

폴리닉 :  아니…… 애초에 이거 라이타니엔 말은 맞아? 현대 라이타니엔어에 이렇게 난해한 발음은 없다고 알고 있는데……

스즈란 :  ……그래도, 듣기는 좋네요.
스즈란 :  처음 로도스 아일랜드 함교에서 대지를 바라봤을 때처럼, 아득하고…… 오래된……
스즈란 :  이건…… 이건 분명 아츠예요…… 극동의 몇몇 고대 아츠와 비슷해요!














세버린 :  누군지 찾아오게.

마을 주민 :  알겠습니다. 하지만 찾은 다음에는요? 바로 체포할까요?

세버린 :  그래.

마을 주민 :  하지만 섣불리 그랬다간, 누군가에겐 선동할 구실이 될 수도 있습니다.

세버린 :  ……이건 감염자와 비감염자의 관계를 이간질하는 짓을 막기 위해서네. 주도자는 어쩌면, 윈터위습인이 아닐 수도 있어.

스즈란 :  세버린 장관님?! 이, 이게 뭐 하시는 거예요?

세버린 :  ……이건 윈터위습의 민요라네. 오래된 의식이나 장례식 따위에서나 사용되지.
세버린 :  감염자와 비감염자 사이를 이간질하고, 위험인물들을 이끌고 마을의 동력 시설을 파괴한 장본인은 윈터위습인일세.
세버린 :  지금은 관용이 필요한 때가 아니지. 난 이 노래가 그들이 주고받는 신호라고 생각하네만…
세버린 :  그들은 자신의 모습을 드러내지 않고 비밀스럽게 활동하고 있을 게야. 게다가 그들 대부분은 캐스터들일 테니, 반드시 조심해야 하네.

폴리닉 :  ……윈터위습인이 대체 뭔데 그러죠?

세버린 :  이곳에 살던 카프리니 족일세. 수백 년 전까지만 해도, 이 산맥의 주인은 그들이었네만…

폴리닉 :  그런데요?

세버린 :  ……그런데 월루몽드와 윈터위습인 사이에 충돌이 발생했고, 결국 이렇게 됐지.

폴리닉 :  아뇨. 그게 다가 아닐 텐데요.
폴리닉 :  컬럼비아도 림 빌리턴도, 이런 역사 하나쯤은 다 있잖아요? 영토 분쟁 없는 나라가 세상에 어디 있다고 그래요?
폴리닉 :  자꾸 어물쩡 대화 주제를 넘기려 하지 마세요. 이렇게 단순하다고요? 그냥 평범한 갈등과 충돌이었다?

세버린 :  ……당신들에게 이런 얘기를 해도 될지 모르겠군. 이건 로도스 아일랜드가 알아도 되는 범주를 넘어선 부분이라서……
세버린 :  일개 헌병이 이런 일을 외부 기업에 섣불리 발설했다간, 몇몇 귀족의 심기가 많이 불편해질 걸세.

폴리닉 :  귀족……? 심기??


폴리닉 :  ……아트로는 당신들을 돕다 희생됐어요! 지금 체면 따위가 중요하단 말인가요?!


스즈란 :  포, 폴리닉 언니! 아트로 언니의 희생이…… 또 다른 싸움의 구실이 되어선 안 돼요!
스즈란 :  세버린 장관님도……! 일단은 수색을 멈춰 주시겠어요? 지금은 이대로라면 다들 감정만 상하고, 일도 돌이킬 수 없게 될 거예요!

폴리닉 :  ……
폴리닉 :  ……아니, 유일한 단서가 눈앞에 있는데, 놓칠 순 없지.

스즈란 :  네……?

폴리닉 :  세버린 씨, ……라이타니엔에는 수많은 이동도시가 있죠. 그 높으신 분들이 과연 당신들 생사 따위에 신경이나 쓸까요?
폴리닉 :  대열곡 사건이 터졌을 때도 마찬가지예요. 가까운 이동도시에서 조금이라도 지원을 나와줬다면, 과연 지금 같은 기근이 발생했을까요?
폴리닉 :  세버린 씨, 지금 당장 당신을 돕고 있는 게 누군지 똑똑히 보라고요!

세버린 :  알고 있네…… 오퍼레이터 폴리닉. 누구보다 잘 알고 있지.
세버린 :  하지만, 로도스 아일랜드가 우릴 평생 책임지지는 못한다는 사실도 똑똑히 알고 있네. 사건이 해결되고 월루몽드가 제자리로 돌아갔을 때, 난 윗선에서 편지를 보내오는 상황은 피하고 싶네.
세버린 :  ……이해해주시게나.

스즈란 :  하지만 폴리닉 언니, 우리는……










스즈란 :  ……폭발?! 또 습격일까요?

마을 주민 :  장관님, 장관님! 도와주십시오! 도움이 필요합니다!

세버린 :  침착해. 무슨 일인가?

마을 주민 :  그놈들, 놈들이 도시 방어 장치를 활성화했습니다! 폭발에, 불꽃에, 놈들이……!

세버린 :  진정해!


타쟈나 :  장관님! 가, 감염자들이 츠벨프톤 거리를 순찰하던 헌병들을 모두 붙잡았습니다! 연락이 끊겼어요!
타쟈나 :  감염자들은 축음기를 사용해 현재 완강히 저항 중입니다. 거리 전체가 이미 통제 불능 상태에요……!

세버린 :  ……그 녀석들이 이렇게 일사불란하게 움직일 리 없는데, 대체 누가 지휘하는 거지?

타쟈나 :  모르겠습니다…… 하지만 시위에 동참하는 비감염자들도 늘어나고 있어요! 월루몽드는 이미…… 반으로 분열됐습니다!

세버린 :  쳇, 윈터위습 놈들…… 제법이군.
세버린 :  이제부턴 거리 전체를 포위하고 사람들을 통제해라. 거리를 되찾아야……
세버린 :  ……크억, 콜록콜록!

타쟈나 :  장관님!

세버린 :  젠장…… 일단 너는 의사당으로 돌아가서 경보를 울리거라. 이젠 단순한 감염자와의 분쟁 수준이 아니야……
세버린 :  지금부터 월루몽드는 계엄령을 선포한다. 모든 주민들을 집합시켜라.

타쟈나 :  하지만! 저희끼리 싸움을 벌였다간, 월루몽드는 이번 겨울조차 버티지 못할……

세버린 :  이건 명령이다!

타쟈나 :  아, 알겠습니다!

세버린 :  ……

세버린 :  로도스 아일랜드의 두 사람…… 당신들은 스스로 옳다고 생각하는 일을 하시게.
세버린 :  솔직히, 이젠 당신들을 신경 쓸 겨를도 없어졌어.

폴리닉 :  아직 정보를 다 듣지 못……

스즈란 :  하, 하지만 지금은 구조에 협력하는 것이 먼저 아닐까요…… 무고한 사람들까지 휘말려 버렸으니……

폴리닉 :  ……이 일만 끝나면, 꼭 범인을 찾을 거야.

세버린 :  그래…… 이 일만 끝나면, 월루몽드는 로도스 아일랜드에 만족할 답을 주겠네. 내 약속하지.

폴리닉 :  그 약속, 꼭 지키시길 바랍니다.

세버린 :  그때까지 내가 살아만 있다면야.








스즈란 :  ……폴리닉 언니, 아무래도 방금 세버린 장관님을 막는 편이 좋지 않았을까요.

폴리닉 :  소용없어. 리사도 봤잖아. 다들 감정이 격해져 있는 상태라고.

스즈란 :  그런가요……?
스즈란 :  하지만 장관님 말씀대로 기근이 코앞에 닥쳤다면, 우리가……

폴리닉 :  ……범인을 빨리 잡는 수밖에 없어. 그래야만 이 소란도 가라앉을 거야. 이 단서를 놓치면, 상황은 더 어려워질 거야.

스즈란 :  전 그런 뜻이 아니었는데……
스즈란 :  ……
스즈란 :  폴리닉 언니, 정말 범인만 잡으면 문제가 해결될 거라고 생각하세요?

폴리닉 :  난……

스즈란 :  ……폴리닉 언니, 진정해요.

폴리닉 :  ……스즈란?

스즈란 :  폴리닉 언니 말이 맞아요…… 지금은 다들 분노에 잠겨서 사리분별을 잘 못하는 거겠죠.
스즈란 :  제 생각에는…… 물론 제 생각일 뿐이지만, 제 감은 엄청 정확하니까요!
스즈란 :  제 생각엔…… 뭔가 더 끔찍한 일이 이미 시작된 것 같아요. 뭐랄까, 옳고 그름을 가리기 힘든 그런 일이……
스즈란 :  우리가 지금 해야 할 일은, 무고하게 휘말린 주민들을 최대한 구하는 거예요.
스즈란 :  폴리닉 언니…… 세버린 장관님을 막지 않으면, 일이 정말 최악의 방향으로 흘러갈지도 몰라요.

폴리닉 :  ……그럴지도 모르지.

스즈란 :  그런데도……

스즈란 :  ……네.
